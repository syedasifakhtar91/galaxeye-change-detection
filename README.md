# GalaxEye Change Detection Project

This project performs satellite image change detection using deep learning segmentation techniques.

## Model Architecture
- U-Net
- ResNet34 Encoder

## Framework
- PyTorch

## Training Details
- Epochs: 1
- Loss Function: BCEWithLogitsLoss
- Optimizer: Adam
- Image Size: 256x256

## Evaluation Metrics
- Precision: 0.7457
- Recall: 0.5254
- F1 Score: 0.6002
- Mean IoU: 0.1239
